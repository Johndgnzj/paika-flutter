# 🚀 快速開始指南

## 立即使用（3 步驟）

### 方法 1：直接開啟網頁（最簡單）
```bash
open build/web/index.html
```
或雙擊 `build/web/index.html` 檔案

### 方法 2：本地伺服器（推薦）
```bash
cd build/web
python3 -m http.server 8000
```
然後在瀏覽器開啟：http://localhost:8000

### 方法 3：開發模式（可修改程式碼）
```bash
flutter run -d chrome
```

## 📖 基本操作流程

### 1️⃣ 開始新局
1. 首頁點擊「開始新局」
2. 選擇底台設定（例如：50×20）
3. 設定 4 位玩家名稱和圖示
4. 點擊「開始打牌」

### 2️⃣ 記錄分數
1. **胡牌（放槍）**：
   - 點擊贏家玩家卡片
   - 選擇「胡牌（放槍）」
   - 選擇放槍者
   - 輸入台數和花牌
   - 確認

2. **自摸**：
   - 點擊贏家玩家卡片
   - 選擇「自摸」
   - 輸入台數和花牌
   - 確認

3. **詐胡**：
   - 點擊詐胡者卡片
   - 選擇「詐胡」
   - 確認

### 3️⃣ 其他功能
- **電子骰子**：點擊右上角骰子圖示
- **還原上局**：右上角選單 → 還原上局
- **換位置**：右上角選單 → 換位置
- **結束牌局**：右上角選單 → 結束牌局

## 💡 使用技巧

### 快速記分
- 常用台數有快速按鈕（2, 4, 6, 8, 10, 12, 16 台）
- 花牌預設為 0，可手動調整
- 自摸會自動加 1 台（依設定）

### 查看分數
- 玩家卡片即時顯示累計分數
- 正分顯示綠色，負分顯示紅色
- 莊家有紅點標示，連莊會顯示次數

### 歷史紀錄
- 首頁顯示最近牌局
- 點擊可查看詳細資料（未來版本）

## 🆘 常見問題

### Q: 如何修改已記錄的分數？
A: 使用「還原上局」功能，然後重新記錄

### Q: 可以中途暫停嗎？
A: 可以！系統會自動儲存，下次開啟會顯示「繼續遊戲」

### Q: 如何清除舊紀錄？
A: 目前需要手動清除瀏覽器本地儲存（未來版本會加入管理功能）

### Q: 支援哪些瀏覽器？
A: Chrome, Safari, Firefox, Edge 最新版本

### Q: 可以在手機上使用嗎？
A: 可以！網頁版支援響應式設計，手機瀏覽器即可使用
